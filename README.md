# Clothing-Swap-and-Exchange
Clothing Swap and Exchange is a web-based platform that allows users to list, exchange, and discover pre-owned clothing items , reducing waste and encouraging sustainable fashion practices.

In My Website --->>  After logging in through the Signup/Login page, the user will be redirected to the Clothing Swap & Exchange page. The account created during registration is used across all categories, including Men's Wear, Women's Wear, and Shoes.

Users can browse products and view detailed information by clicking the View Details button. Products can be added to the cart using the Add to Cart option. Once added, the products will appear in the My Orders section, where users can select a delivery address and place an order.

Users also have the option to remove products from their cart/order list. Additionally, the My Orders page displays the user's profile information.
